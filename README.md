# My - Twitter Bot

> Help you when you are feeling lonely on twitter, never leave behind

## Run

```sh
find . -type f -name 'selenium*.png' -delete
find . -type f -name '*.log' -delete
kill -9 (pgrep firefox); clear;robot --variablefile vars.yaml twitter.robot
```
